Function getUnifiedERC20Balance

  • getUnifiedERC20Balance(parameters): Promise<UnifiedERC20Balance>

  • Fetches and aggregates ERC20 token balances across multiple chains for a given account

    Parameters

    • parameters: GetUnifiedERC20BalanceParameters

      GetUnifiedERC20BalanceParameters Configuration for balance fetching

    Returns Promise<UnifiedERC20Balance>

    Promise resolving to UnifiedERC20Balance containing total balance and per-chain breakdown

    Throws

    Error if token decimals mismatch across chains

    Example

    const balance = await getUnifiedERC20Balance({
      mcToken: mcUSDC,
      account: myMultichainAccount
    });

    console.log(`Total balance: ${balance.balance}`);
    console.log(`Decimals: ${balance.decimals}`);
    balance.breakdown.forEach(chainBalance => {
      console.log(`Chain ${chainBalance.chainId}: ${chainBalance.balance}`);
    });
