Function createSmartAccountClient

  • createSmartAccountClient<TSmartAccount, TTransport, TChain>(parameters): SmartAccountClient<ENTRYPOINT_ADDRESS_V07_TYPE, TTransport, TChain, TSmartAccount>

  • Creates a EIP-4337 compliant Bundler Client with a given Transport configured for a Chain.

    • Docs: https://docs.pimlico.io/permissionless/reference/clients/smartAccountClient

    A Bundler Client is an interface to "erc 4337" JSON-RPC API methods such as sending user operation, estimating gas for a user operation, get user operation receipt, etc through Bundler Actions.
