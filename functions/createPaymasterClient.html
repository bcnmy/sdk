Function createPaymasterClient

  • createPaymasterClient<transport, chain>(parameters): PaymasterClient<undefined | Chain>

  • Creates a Paymaster Client with a given Transport configured for a Chain.

    A Paymaster Client is an interface to "paymaster endpoints" JSON-RPC API methods such as sponsoring user operation, etc through Paymaster Actions.
